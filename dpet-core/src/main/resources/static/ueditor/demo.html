01.身体影响心理。当我们表现出快乐的、充满信心与活力的表情、动作时，我们的心情也就会随之改变。如果遇上情绪烦恼，可以考虑睡在床上“度假”一天，这样可以松弛紧张的压力和忧郁的情绪，解除烦闷的感觉。 02.人生中93%的烦恼都是自找的，因为他们只存在于自己的想象中，根本就不会出现。烦恼就像一张无形的大网，藏在人们心里，一不留神它就会出来网走我们的快乐和好心情，所以千万不要自寻烦恼，也不用今天就急着解决明天的难题。 03.挫折如果已经发生，那就应当面对它，寻找解决的办法；如果挫折已经过去，那就应当丢开它，不要老是把它保留在记忆里。 04.高情商者与低情商者的一个重要的区别，就在于高情商者要比低情者更善于处理情感问题。因而，情商的一个重要组成部分就是处理情感的能力。 05.享受自己的生活，不要与别人作比较。有些事情虽然无法改变，但你可以改变自己的心态。如果总是羡慕别人、看轻自己，那么人生将是何等乏味与痛苦。能乐于接受自己、肯定自己的人，才会得到快乐。 06.按照自己的意愿去做人做事，我们就不必勉强改变自己，不必费心掩饰自己。这样，就能少一些精神的束缚，多几分心灵的舒展，就能少一点不必要的烦恼，多几分人生的快乐与轻松。 07.信心是一种精神状态，是一个人对宇宙力量的一种了解、信任和融合的表现。通过调整你的内心世界，接受来自于无限智慧的力量，你可以获取自信心。拥有自信心，你就可以消除一切消极因素，克服疑惑和恐惧，消除自卑心理，并将其转化为积极的、实质性的东西。 08.退缩会养成一个人怯懦、消极的习惯。退缩是一种恐惧心理，具有传染性。退缩使自我失去平稳。只要改掉退缩的坏习惯，积极地去追求、去奋斗，你出色的表现会让自己都吃惊，“现实的我”将更趋近个人心中“理想的我”。 09.真正的人生需要挫折。遇到逆境就一味消沉的人，是肤浅的；一有不顺心的事就惶惶不可终日的人，是脆弱的。一个人不懂得人生的艰辛，就容易傲慢和骄纵。未尝过人生苦难的人，也往往难当重任。 10.人不应该把自己降为感情的奴隶。不应把全盘的生命计划，重要的生命问题，都去同感情商量。无论你周遭的事情是怎样的不顺利，你都应笑对逆境，努力去支配你的环境，把你自己从不幸中振作起来。你应背向黑暗，面对光明。这样阴影自会留在你的后面！ 11.当你遇到怄气的事时，发发牢骚可以适度缓解心中的怒气，避免不必要的压力积累，对健康是有益的。但是，发牢骚也得注意把握好一个度，要注意频繁程度，要注意场合，要看对象。 12.哭，是一种有效的解除紧张、烦恼与痛苦情绪的方法，尤其是对突如其来的打击所造成的高度紧张、极度痛苦，可以起到缓解作用。 13.快乐很简单，简单就是快乐，随意就是快乐，平平淡淡就是快乐，其实，生活中并不缺乏快乐，只是缺乏自觉发现快乐的眼睛和感受快乐的心。 丢掉所有的不快乐，就是快乐。其实快乐离我们并不遥远，不必刻意去追求，它其实就在我们的身边，在无意中就能发现它。 14.在现实生活中，终日烦恼的人，实际上并不是遭遇了多大的不幸，而是根源于烦恼者的内心世界。因此，当烦恼降临的时候，我们既不要怨天尤人，也不要自暴自弃，要学会给心灵松绑，从心理上调适自己，避免烦恼成心病。 15.快乐之道无它，对那些自己无法改变或力所不能及的事情，要抱着拿得起，放得下的态度，不去忧虑，或者创造出另一种情境，或者采取迂回的办法自我转化，把自己的情感和精力转移到其他活动中去，使自己没有时间和可能沉浸在这种烦恼之中。
